== Description ==

Using this plugin you can create stylized alert, download, info, comment, search, tip, twitter, tool and note boxes. All just by remembering simple short codes. 

Demo: http://icustomizethesis.com/demo-style-boxes-plugin-with-short-code/

== Installation ==

To install the plugin follow these steps :

1. Download the style-box-with-short-code.zip file to your local machine.
2. Unzip the file 
3. Upload "style-box-with-short-code" folder to the "/wp-content/plugins/" directory
4. Activate the plugin through the 'Plugins' menu in WordPress

That's it!

== Frequently Asked Questions ==

How to create an Alert box? 
In your WP post editor, use short code [alert] your text [/alert]
That's it, your text will appear in a stylized alert box. 

Available short codes are: 

[note]your text [/note]
[alert]your text [/alert]
[info]your text [/info]
[tip]your text [/tip]
[down]your text [/down]
[finder]your text [/finder]
[twitter]your text [/twitter]
[tool]your text [/tool]
[talk]your text [/talk]

New short codes:

[help]your text [/help]
[linux]your text [/linux]
[windows]your text [/windows]
